# Known Limitations — QML Delivery

| Limitación | Impacto |
|---|---|
| Audio físico no probado en todas las configuraciones | Posible falta de audio en algunas salidas |
| Módulos experimentales no visibles en delivery mode | No afecta flujo core |
| Biblioteca sin indexación automática en QML | Requiere refrescar manualmente |
| Playlists sin import/export M3U | Funcionalidad básica disponible |
| Smart Tagging sin aplicación automática | Solo vista de sugerencias |
| EQ sin backend DSP completo | Interfaz visual sin procesamiento real |
| Library Doctor sin reparación automática | Solo diagnóstico |
| Disc Lab sin ripping real | Solo detección de unidad |
